Check-in

O sistema de Check-in permite registrar a presença dos alunos de forma rápida e eficiente. Existem duas maneiras principais de realizar o check-in:

1. Escanear QR Code

Esta é a forma mais rápida. O aluno apresenta seu QR Code (disponível no app do aluno) para a câmera do dispositivo.

  • Acesse a página de Check-in.
  • Permita o acesso à câmera se solicitado.
  • Aponte a câmera para o QR Code do aluno.
  • O sistema identificará o aluno e registrará a presença automaticamente.

2. Busca Manual

Caso o aluno não esteja com o QR Code, você pode buscá-lo manualmente.

  • No campo "Busca Manual", digite o Nome, E-mail, CPF ou ID da Venda.
  • Clique no botão de busca (lupa) ou pressione Enter.
  • Se o aluno for encontrado, seus dados aparecerão na tela para confirmação.

Dica: Mantenha o ambiente iluminado para facilitar a leitura do QR Code.
